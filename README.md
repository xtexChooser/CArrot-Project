1. 本项目所涉及的时间均为北京时间(或Asia/Shanghai, UTC+8:00)
2. [CA(命令助手)](https://gitee.com/projectxero/ca): 命令助手是一款用于辅助玩家输入Minecraft(基岩版)命令的脚本
